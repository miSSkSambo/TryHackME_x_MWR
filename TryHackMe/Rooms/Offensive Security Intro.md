## 📌 Overview

This room introduces the basics of **Offensive Security**, where you think like a hacker to identify vulnerabilities before malicious attackers do.

You get hands-on experience by hacking a **fake banking website (FakeBank)** in a safe and legal environment. It demonstrates how ethical hackers discover hidden pages and exploit weak security controls.

---

## 🎯 Learning Objectives

- Understand what Offensive Security is
- Learn how hackers think and approach systems
- Discover hidden web pages using tools
- Exploit a vulnerable web application
- Gain hands-on experience with basic web hacking

---

## 🧠 Key Concepts

- **Offensive Security**
- **Web Application Testing**
- **Directory Brute Forcing (Dirb)**
- **Hidden Endpoints**
- **Basic Exploitation**
- **Ethical Hacking**

## 🚀 Summary

This room provides a beginner-friendly introduction to **ethical hacking** by showing how attackers:

- Discover hidden pages using tools like **dirb**
- Exploit poor access control
- Manipulate web applications to gain unauthorized benefits

It highlights how small security oversights (like exposed admin pages) can lead to major vulnerabilities.
