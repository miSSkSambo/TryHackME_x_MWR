# 🔍 Intro to Digital Forensics  
🔗 Link to the room: https://tryhackme.com/room/introdigitalforensics 

---

## 📌 Overview  
This room introduces **Digital Forensics**, the process of collecting, preserving, and analyzing digital evidence for investigations.

It covers:
- Digital forensics fundamentals  
- Evidence handling procedures  
- Chain of custody  
- Metadata analysis  
- EXIF data investigation  

---

## 🎯 Learning Objectives  
- Understand digital forensics processes  
- Learn proper evidence handling techniques  
- Analyse document metadata  
- Extract EXIF data from images  
- Use forensic tools like `pdfinfo` and `exiftool`
