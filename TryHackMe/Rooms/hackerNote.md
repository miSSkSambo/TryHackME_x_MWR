## 📖 Description

The **HackerNote TryHackMe room** focuses on web application security, specifically demonstrating how poor backend logic can lead to vulnerabilities like **username enumeration**, **timing attacks**, and weak authentication mechanisms.

It also covers:
- Creating custom wordlists  
- Brute-force attacks using Hydra  
- API exploitation  
- Privilege escalation using a real-world CVE
