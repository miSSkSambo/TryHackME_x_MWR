---

link  to the room: https://tryhackme.com/room/25daysofchristmas

## 📌 Overview

This room is a **25-day beginner-friendly cybersecurity challenge series** where you help Santa’s company recover from a cyber attack by the **Christmas Monster**.

Each day introduces a new topic, including:
- Linux Security
- Web Application Security
- Network Security
- Forensics
- Reverse Engineering

---

## 🎯 Learning Objectives

- Gain hands-on cybersecurity experience
- Learn fundamental attack and defense techniques
- Understand real-world vulnerabilities
- Build problem-solving skills through CTF challenges

---

## 🧠 Key Concepts

- **Web Exploitation (SQLi, LFI, XSS)**
- **Network Analysis (PCAP, Nmap)**
- **Privilege Escalation**
- **OSINT**
- **Cryptography**
- **Brute Force Attacks**
- **Reverse Engineering**
- **Cloud Security (S3 Buckets)**
