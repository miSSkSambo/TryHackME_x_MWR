## 📌 Task 7: Handling Data With Pandas

**Q: What are the two main types of data within Pandas?**  
**A:** Series and Dataframes  

**Q: What is the name of the Pandas function that reads a CSV file?**  
**A:** read_csv  

**Q: Name the Pandas function you would use if you only wanted to display the first few rows**  
**A:** head  

**Q: Name the Pandas function you would use if you only wanted to display the last few rows**  
**A:** tail  

**Q: What Pandas function will give you a numerical count of the amount of columns and rows the dataset contains?**  
**A:** shape
