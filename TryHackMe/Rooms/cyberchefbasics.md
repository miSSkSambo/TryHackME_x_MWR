## 📌 Overview

This room introduces **CyberChef**, a powerful web-based tool often called the *“Swiss Army knife of cybersecurity.”* It allows users to perform a wide range of data transformations such as encoding, decoding, encryption, and extraction.

The room focuses on:
- Understanding how CyberChef works
- Navigating its interface (Operations, Recipe, Input, Output)
- Performing common encoding/decoding tasks (Base64, URL, Base85, etc.)
- Extracting useful information like IP addresses, emails, and domains
- Converting data formats such as timestamps and binary

---

## 🎯 Learning Objectives

- Learn how to use CyberChef effectively
- Understand encoding formats (Base64, URL encoding, Base85)
- Practice extracting useful data from raw input
- Convert between different data formats
- Apply problem-solving using CyberChef recipes

- **Encoding vs Decoding**
- **Base64 / Base85**
