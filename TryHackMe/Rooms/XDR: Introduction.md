--- 
link to the room: https://tryhackme.com/room/xdrintroduction

## 📌 Overview

This room introduces **Microsoft Defender XDR (Extended Detection and Response)**, a unified security platform that integrates data from multiple sources to improve threat detection, investigation, and response.

Unlike traditional tools that operate in isolation (like EDR), XDR provides:
- Centralized visibility
- Cross-domain threat detection
- Automated response capabilities

---

## 🎯 Learning Objectives

- Understand what XDR is and how it differs from EDR
- Learn about Microsoft Defender XDR and its components
- Understand how XDR correlates signals across services
- Learn role-based access control (RBAC)
- Navigate the Microsoft Defender portal
