## 📖 Description

The **Toolbox: Vim TryHackMe room** introduces the powerful text editor **Vim**, widely used in cybersecurity and system administration. It teaches efficient text editing, navigation, and command usage, making it an essential tool for working in terminal environments.

The room covers:
- Vim modes (Insert, Command, Visual)  
- Navigation and text editing  
- Saving and exiting files  
- Copy, cut, and paste operations  
- Searching and replacing text

## 🧪 Full Questions & Answers

**Q: Install Vim**  
**A:** No answer needed  

**Q: Launch Vim**  
**A:** No answer needed  

---

**Q: How do we enter INSERT mode?**  
**A:** i  

**Q: How do we start entering text?**  
**A:** typing  

**Q: How do we return to command mode?**  
**A:** esc  

---

**Q: Move cursor left?**  
**A:** h  

**Q: Move cursor right?**  
**A:** l  

**Q: Move cursor up?**  
**A:** k  

**Q: Move cursor down?**  
**A:** j  

---

**Q: Jump to start of a word?**  
**A:** w  

**Q: Jump to end of a word?**  
**A:** e  

---

**Q: Insert before cursor?**  
**A:** i  

**Q: Insert at beginning of line?**  
**A:** I  

**Q: Append after cursor?**  
**A:** a  

**Q: Append at end of line?**  
**A:** A  

**Q: New line below current line?**  
**A:** o  

---

**Q: Write file without exiting?**  
**A:** :w  

**Q: Write file as root?**  
**A:** :w !sudo tee %  

**Q: Write and quit?**  
**A:** :wq  

**Q: Quit?**  
**A:** :q  

**Q: Force quit?**  
**A:** :q!  

**Q: Save and quit all tabs?**  
**A:** :wqa  

---

**Q: Copy a line?**  
**A:** yy  

**Q: Copy 2 lines?**  
**A:** 2yy  

**Q: Copy to end of line?**  
**A:** y$  

---

**Q: Paste after cursor?**  
**A:** p  

**Q: Paste before cursor?**  
**A:** P  

---

**Q: Cut a line?**  
**A:** dd  

**Q: Cut 2 lines?**  
**A:** 2dd  

**Q: Cut to end of line?**  
**A:** D  

**Q: Cut a character?**  
**A:** x  

---

**Q: Search forward?**  
**A:** /pattern  

**Q: Search backward?**  
**A:** ?pattern  

**Q: Repeat search same direction?**  
**A:** n  

**Q: Repeat search opposite direction?**  
**A:** N  

---

**Q: Replace "old" with "new"?**  
**A:** :%s/old/new/g  

**Q: Use grep in Vim?**  
**A:** :vimgrep
