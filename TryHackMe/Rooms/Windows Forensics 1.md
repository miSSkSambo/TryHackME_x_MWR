## 📖 Description

The **Windows Forensics 1 TryHackMe room** introduces the fundamentals of **Windows Registry Forensics**, a key area in digital forensics used to investigate user activity and system behavior.

The room focuses on:
- Understanding the Windows Registry structure  
- Identifying forensic artifacts  
- Extracting and analyzing registry hives  
- Investigating user activity, executed programs, and connected devices  
- Performing real-world forensic analysis using tools like Registry Explorer

## 🧠 Key Takeaways

- Windows Registry is a **goldmine of forensic artifacts**  
- User activity can be reconstructed from registry data  
- Multiple artifacts track execution (UserAssist, AmCache, BAM)  
- USB and device history can be recovered  
- Timezone and system info are critical for timelines  
- Forensic tools simplify analysis but require understanding
