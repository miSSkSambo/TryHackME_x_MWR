**Q:** What process focuses on collecting data to analyse the performance and stability of services?  
**A:** Monitoring

## 🔹 Task 4: Shifting Left

**Q:** What term is used to describe accounting for security from the earliest stages in a development lifecycle?  
**A:** shift left  

**Q:** What is the development approach where security is introduced from the early stages of a development lifecycle until the final stages?  
**A:** DevSecOps

# 🚀 Summary

This room explains how software development evolved:

- **Waterfall** → rigid, slow, poor communication  
- **Agile** → flexible, collaborative  
- **DevOps** → automation + integration + culture  

Then introduces:

- **DevSecOps** → security integrated from the start  

---

## ⚠️ Key Takeaways

- Security must be integrated early (**Shift Left**)  
- DevOps improves speed, but needs security  
- DevSecOps = **culture + automation + shared responsibility**  
- Communication and trust are critical  

---

## 🔐 Core Concepts

- CI/CD → automated testing & deployment  
- IaC → infrastructure via code  
- Monitoring → system visibility  
- Orchestration → automated workflows  

---

## 💡 Final Insight

DevSecOps is not just tools — it is:
➡️ A **culture shift**  
➡️ A **shared responsibility**  
➡️ A **continuous security mindset**
