## 📌 Overview

This room introduces **fundamental networking concepts**, including how data travels across networks and how systems communicate.

It covers:
- The **OSI Model**
- The **TCP/IP Model**
- IP addressing and subnets
- TCP vs UDP communication
- Encapsulation and packet flow
- Basic network interaction using **telnet**

## 🚀 Summary

This room builds a strong foundation in networking by explaining how systems communicate.

Key takeaways:
- The **OSI model** helps understand network communication layers
- The **TCP/IP model** is the real-world implementation
- **IP addresses** uniquely identify devices
- **TCP** is reliable; **UDP** is faster but unreliable
- Data travels through networks using **encapsulation**
- Tools like **telnet** allow direct interaction with services
