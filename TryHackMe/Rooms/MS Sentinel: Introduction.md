---
link to the room: https://tryhackme.com/room/sentinelintroduction

## 📌 Overview

This room introduces **Microsoft Sentinel**, a cloud-native security solution used by Security Operations Centers (SOC) to monitor, detect, investigate, and respond to threats.

It explains how Sentinel combines:
- **SIEM (Security Information and Event Management)**
- **SOAR (Security Orchestration, Automation, and Response)**

to provide a complete security operations platform.

---

## 🎯 Learning Objectives

- Understand the role of a SOC Analyst
- Learn what Microsoft Sentinel is
- Understand SIEM and SOAR concepts
- Learn how Sentinel works (Collect → Detect → Investigate → Respond)
- Identify when to use Microsoft Sentinel

## 🚀 Summary

Microsoft Sentinel is a **powerful cloud-native SIEM + SOAR platform** that helps SOC teams:

- Collect and analyze security data

- Investigate incidents efficiently

It provides a **centralized, scalable, and intelligent security solution** for modern organizations.
