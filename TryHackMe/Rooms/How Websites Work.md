# 🌐 How Websites Work – Q&A Notes

---

## 🎯 What This Room Is About

This room introduces the fundamentals of how websites work, including how browsers communicate with servers and how web pages are built using HTML, CSS, and JavaScript.
link to the room : https://tryhackme.com/room/howwebsiteswork


It also introduces basic web security concepts, such as:
- Sensitive Data Exposure  
- HTML Injection  

The goal is to understand both how websites function and how they can be exploited if not properly secured.

---

## 📘 Task 1: How Websites Work

**Q:** What term best describes the part of a web application that is rendered and displayed in your browser?  
**A:** Front End

## 📘 Task 2: HTML

**Q:** When fixing the broken image on the cat website, what hidden text is revealed?  
**A:** HTMLHERO  

**Q:** After adding a dog image using an `<img>` tag with the source `img/dog-1.png`, what text appears in the image?  
**A:** DOGHTML
