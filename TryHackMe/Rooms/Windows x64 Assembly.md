The **Windows x64 Assembly TryHackMe room** introduces the fundamentals of **reverse engineering and low-level programming**. It explains how computers operate at a lower level using assembly language, covering concepts such as number systems, registers, memory layout, and calling conventions.

This room builds the foundation required for:

## 🧠 Key Takeaways

- Assembly is the **low-level language** executed by CPUs  
- Registers act like **fast variables inside the CPU**  
- Memory is divided into sections (stack, heap, etc.)  
- Binary operations are fundamental to computation  
- Calling conventions define how functions interact  
- Reverse engineering requires understanding how code runs internally
