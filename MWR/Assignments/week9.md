## 📌 Overview  
This week focuses on **File Upload Vulnerabilities** — a critical web security issue that can lead to **Remote Code Execution (RCE)** and full server compromise.

You will learn:
- Arbitrary file upload vulnerabilities  
- File overwrite attacks  
- Web shells and reverse shells  
- Client-side vs server-side filtering  
- Bypassing upload restrictions  
- Real-world exploitation techniques
