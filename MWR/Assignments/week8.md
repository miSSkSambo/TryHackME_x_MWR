## 📌 Overview  
This week focuses on **SQL Injection (SQLi)** — one of the most critical web vulnerabilities.

You will learn:
- Database fundamentals  
- SQL query structure  
- SQL Injection types (In-Band, Blind, Out-of-Band)  
- Authentication bypass  
- Filter evasion techniques  
- Real-world exploitation scenarios  

---

## 🎯 Learning Objectives  
- Understand relational databases and SQL  
- Identify SQL Injection vulnerabilities  
- Perform exploitation techniques  
- Bypass filters and protections  
- Extract sensitive data from databases  
- Understand secure coding practices  

---

## 🧠 Key Concepts  
- DBMS & RDBMS  
- SQL Queries (SELECT, INSERT, UPDATE, DELETE)  
- SQL Injection  
- UNION Attacks  
- Blind SQLi  
- Second-Order SQLi  
- Filter Evasion  
- Out-of-Band (OOB) SQLi
