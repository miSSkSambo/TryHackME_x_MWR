### 🔹 Task 2: What is SQL Injection?  

**Q:** What is the acronym for the software that controls a database?  
**A:** DBMS  

**Q:** What is the most common type of database control software?  
**A:** RDBMS  

**Q:** What is the name of the grid-like structure which holds the data?  
**A:** Table  

**Q:** Where is information about the database structure stored in SQL?  
**A:** INFORMATION_SCHEMA  

**Q:** What does SQL stand for?  
**A:** Structured Query Language  

**Q:** What SQL statement is used to add data?  
**A:** INSERT  

**Q:** What SQL statement is used to round up?  
**A:** Ceiling  

**Q:** What character signifies the end of an SQL query?  
**A:** ;  

**Q:** What is the most common SQL query?  
**A:** ' or 1=1 --
